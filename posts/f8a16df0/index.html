macOS Sierra 睡眠方式設置

發表於 2017-02-03 | 分類於 IT | | 閱讀次數

macOS 通過pmset命令設置hibernatemode的值來調整系統睡眠方式。

以下內容翻譯自pmset命令的幫助文件（可以運行man pmset查看）

macOS 安全休眠說明

hibernatemode 可以設置的值有 0、3和25。macOS 是否寫入休眠映像文件到硬盤也同時取決于 standby 和 autopoweroff 的值。

例如，對於支持休眠的台式電腦將在 standby 設置的時間之後寫入休眠映像文件。要完全禁止休眠映像，請確保 hibernatemode，standby 和 autopoweroff 的值都要設置為 0。

hibernatemode = 0 是桌面電腦的默認設置。系統不會將內存備份到硬盤上。系統必須從內存中的內容恢復；如果掉電，將丟失內容。這是一種舊的睡眠設置。

hibernatemode = 3 是便攜式電腦的默認設置。系統將存儲一份內存的副本到硬盤上，並將保持內存的供電。系統將會從內存恢復，除非掉電，迫使系統從休眠映像恢復。

hibernatemode = 25 只能通過pmset命令來設置。系統將存儲一份內存的副本到硬盤上，並將切斷內存的供電。系統將從硬盤映像恢復。如果你想以較慢的速度睡眠、較慢的速度恢復和更長的電池壽命，那麽你可以使用這個設置。

注意休眠文件只能指向位於根目錄所在卷的上的文件。

官方的說明已經很清楚了，我再簡化一下，系統休眠的時候：

  1. 值為 0：不向硬盤寫入文件，內存一直通電，從內存喚醒系統
  2. 值為 3：這個設置還要取決與 standby 的設置。如果內存通電，則從內存喚醒系統；如果內存不通電，則從硬盤文件喚醒系統
  3. 值為 25：內存斷電，從硬盤鏡像喚醒系統

顯然，如果為了hibernatemode = 3這種設備對於MacBook 來說是最好的設置狀態，也是默認的設置狀態，我們一般不需要進行修改。但是有些用戶偏向於更省電，那麽設置成3最好；有些用戶偏向於延長SSD壽命，那麽設置成1最好。
