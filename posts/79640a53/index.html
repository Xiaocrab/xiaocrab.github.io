Windows 7 使用 2 年的心得

發表於 2012-02-03 | 分類於 IT | | 閱讀次數

自 2010 年 2 月 3 日使用 Windows 7 系统以來，已有兩年時間，在此寫一篇文章以紀錄使用心得。

Windows 7 RTM 釋出於 2009 年夏季，現在市場佔有率已經超越 XP。與 Windows XP 相比改進很大，更易於使用，是 Microsoft 最成功的一款桌面作業系統。Windows 7 的伺服器版本——Server 2008 R2與 Windows 7 同核心，僅支援 x64，也比較好用。Windows 7 有 6 個版本，但是常用的只有 4 個版本：家庭基礎版、家庭高階版、專業版、旗艦版。

市場上多數電腦已經預裝 Windows 7，其中大多數系統是 Ghost 系統，採用「軟激活」或「硬激活」方式激活系统。請人裝系統，除非提前說明，絕大多數是 Ghost 系統，10分鐘內就能裝好。但是 Ghost 系統經過精簡，可能會移除一些不常用的功能、植入一些廣告（常見的就是鎖定 IE 瀏覽器主頁），適用於一般以娛樂為主要使用目的人群。若是安裝純淨、功能完全的系統，建議使用系統安裝光碟（映像），適用於高階使用人群。

再談 Windows 7 的激活問題，常用的有 2 種方法

  1. 使用「神 Key」激活：直接輸入密陰就可以激活。此方法需要連結 Internet，相關的神 Key 請到「翱翔-百度空間」某寶網尋找；

  2. OEM 激活：需同時滿足下面三個條件

    1. BIOS中有 SLIC 2.1 表；
    2. 系統中有 OEM 證書（副檔名為 XPM-MS），並與 SLIC 匹配；
    3. 系統中有 SLP Key。

    此方法不需聯網，但是其中任何一個條件有問題，則激活失敗。

    鑑於此，產生了「硬激活」、「軟激活」等激活方法。關於這些激活方法的優缺點請自己研究。如果是品牌機，建議硬激活。

實際上 Windows 7 只要你用正常的激活方法（破解系統、篡改系統除外）將系統激活，就是正版的系統！享受正版系統的待遇（自動更新等）。若是系統沒有激活，就是從 Microsoft 專賣店買來的系統也和網上下載的沒什麼區別。個人建議使用 MSDN 旗艦版 + 硬激活，完美無比，和買來的系統一樣一樣的。

下面再來評評 Windows 7 實用的新功能

  • Windows Virtual PC：這是閹割版的 Microsoft Virtual PC，僅支援運行 XP x86、Vista x86、Windows 7 x86 和 XP Mode，功能少，不推薦使用，建議使用 VMware 系列虛擬機軟體。

  • VHD 支援：Windows 7 原生支援 VHD。支援 VHD 啟動（僅旗艦版支援）、掛載 VHD（僅專業版、旗艦版支援），這是個好東西喲。

  • 觸控式螢幕支援：現在雖然只有很少人家裡有觸控式螢幕，但是我們對觸控式螢幕並不陌生。許多自助終端、手機、平板電腦都是觸屏，如果你身邊有觸控式螢幕要好好試試這個功能。

  • 藍光、USB3.0(Windows 7 SP1) 支援：這些都是新技術，幾年後就會流行的。

  • 64 位元系統逐漸普及：Windows 7 和現在的高配置硬體推動了 64 位元系統的普及，更多的人開始使用 x64 系統。

  • Windows Media Player 12：支援更多的檔案格式播放，介面清爽。

  • More：許可權控制、超級工作列、便捷的網路連接、Aero效果…….許多功能，更易於使用。

使用 2 年以來，感覺它比 XP 好，但是有些功能與 XP 不同。

設定方法不一樣、路徑不一樣。因此，有些人說 Windows 7 不好用，實際上是他們不會用！關於 XP 和 Windows 7 的差別還是有一些的。比如關機、登出、睡眠、休眠等。雖然時不時的會開啟 XP 虛擬機，但是同樣的設定在 XP 上怎麼設定已經有些忘記了。常常不自覺的在 XP 系統裡滑鼠拖動最大化、半視窗最大化可是試過之後才記得 XP 是不支援的。相比於 XP，Windows 7 很少宕機、藍屏，並且開關機速度超快（於同等配置的計算機相比）。

關於 Windows 7 對程式的兼容問題

至今發現只有個別程式不兼容，對程式的兼容不錯！

關於 Windows 7 對硬體的兼容問題

只要不是特別古老的硬體，5、6年內的都可以兼容！
